## Test task

Use docker compose (php 7.4 + mysql 5.7):

- Make sure that 80 port is free or change port for app in docker-compose.yml.
- Run <code>doker-compose up -d</code>
- Run on container *_app  <code>composer install</code>
- Tests. Run on container *_app <code>composer test</code>
